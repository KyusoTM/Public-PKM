# コメント(Java)

## 単一行のコメント
文法 : `//`
```java
// この部分が単一行のコメント
public class Main {
    public static void main(String... args) {
        System.out.println("Hello World");  //文末にも記入可能
    }
}
```

## 複数行のコメント (ドキュメンテーション)
文法 : `/* ... */`
```java
/*
    囲んでいる範囲がコメントして認識
*/
public class Main {
    public static void main(String... args) {
        System.out.println("Hello World");  //文末にも記入可能
    }
}
```
