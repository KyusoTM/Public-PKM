# アクセス修飾子

## javaのアクセス修飾子
アクセス修飾子とアクセスレベル
| アクセス修飾子 | アクセスレベル    | 意味                                     |
| -------------- | ----------------- | ---------------------------------------- |
| public         | public            | どのクラスからもアクセス可能             |
| protected      | protected         | 同一パッケージ or サブクラスからのみ     |
| なし           | パッケージprivate | 同一パッケージ内からのみ                 |
| private        | private           | 自クラスおよび同じクラスの別インスタンス |

※privateなメンバであっても，同じクラスから作成された別のインスタンスから操作可能
