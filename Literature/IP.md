# IP
- Internet Protocol

インターネット上のホストへデータを送信するための通信プロトコル．
- [[OSI基本参照モデル]]のネットワーク層に対応する

ネットワーク上の各ホスト端末にIPアドレスを割り当てることで，指定されたホストにパケットを送信できる．

パケットの到達は保証しないため，その保証のためには[[TCP]]等のプロトコルを適用する必要がある．
