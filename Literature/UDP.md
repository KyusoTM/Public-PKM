# UDP

- User Diagram Protocol

ハンドシェイクを行わずに通信を行う，コネクションレス型の通信プロトコル．

送信側が一方的にデータを送り続ける通信方式であり，データの信頼性を担保しない代わりに遅延が発生しにくい特徴を有する．

## 主な用途
- 動画・音声のストリーミング

## 関連するプロトコル
- [[TCP]]
  - 送受信データの信頼性を担保するプロトコル．
