# WCS

- Warehouse Control System : 倉庫制御システム

## 概要
物流センターにおける[[マテハン]]設備群を運用するための自動制御システム．

操作対象となる主な装置は以下の通り．
- コンベヤ
- ソーター
  - 搬送物の仕分け機器
- [[AGV]]・[[AMR]]

WCSは設備制御に特化しており，作業工程や人員管理には[[WMS]]や[[WES]]との連携が必要．
