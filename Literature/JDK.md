# JDK

- Java Development Kit

## 概要
オラクルが提供する[[Java SE]]開発ツールキットの製品名．

もともとは，[[Java SE]]の開発ツールキット全般を指していたが，[[Oracle]]が商用登録したことで，厳密には[[Oracle Java SE]]をさす用語となっている．

[[Java]]開発用のツールキット開発を行うプロジェクト及び，リポジトリは[[OpenJDK]]の名称で運営されている．
